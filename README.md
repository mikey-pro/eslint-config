## **@mikey-pro/eslint-config**

A preset ESLint configuration for use with the Mikey Pro style guide

<a href="https://www.npmjs.com/package/@mikey-pro/eslint-config"><img src="https://img.shields.io/bundlephobia/min/@mikey-pro/eslint-config?color=9987d8&label=package%20size&logo=ok&style=for-the-badge" alt="Package size badge" width="80px" /></a>

## Usage

### Install

```shell
npm i -D @mikey-pro/eslint-config

yarn add -D @mikey-pro/eslint-config
```

### Configure

Extend to ESLint in `package.json`:

```json
{
  "eslintConfig": {
    "extends": [
      "@mikey-pro/eslint-config"
    ]
  }
}
```
